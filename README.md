# SnakesAndLadders

## Requirements
To use that program you need Python 3.9

To execute test you need pytest. If you do not have it, you can install following the instructionts that you can find into this link:
https://blog.finxter.com/how-to-install-pytest-in-python/


## How to run?
You can run the script from the command-line using the following command from the same folder.
```
python game.py
```

## How to run test?
You can run tests from the command-line using using the following command from the same folder.
```
python -m pytest
```
